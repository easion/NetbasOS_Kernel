Description:
-----------------------------------------------------------------
    JEB Means "Jicama Experimental Build". MicroKernel OS.
    JICAMA OS is Not a Simply UNIX Clone, It Written From Scratch By Deng Pingping. Only Developed For i386 Based IBM Personal Compute and compatible Machine, including multitasking, virtual memory(paging), shared libraries.  Interrupt, Exception and System Call running well, support GNU GRUB BootLoader. It Aims Towards POSIX Compliance, But Not Distributed on GNU GPL.
Project Homepage: http://cosoft.org.cn/projects/jicama

Software you need to compile or running this OS:
        *Tested And Compiled On: Microsoft windows 98, (Use VMWARE Software run it)
	* GNU gcc for windows (DJGPP)version 2.7.2.1 or later - c compiler
	* DJGPP port of GNU Bintuils 2.11.2 with ELF support - tools to link kernel
	* NASM for windows version 0.98 or later - x86 Assembler

Goals:
-----------------------------------------------------------------
sturdiness kernel, Beautiful GUI.  just for fun :)


Supported List:
-----------------------------------------------------------------
-  i386 or Newer Processor and 12MB RAM or more Memory
-  GRUB For Default Bootloader Program, implement multiboot
-  run COFF&PE&ELF format user program
-  NEC Floppy, IDE Harddisk
-  PS/2 Mouse
-  PCI Bus(not debug)
-  vindows GUI system: PCX,BMP,ICO SHOWING, For VBE(VESA 2.0)
-  FAT and Ext2 file system  and more...
